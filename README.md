## Quick Start

Put checkpoints (Per-trained and fine-tuned) in ./home dir as the following structure:

```text
├─checkpoints
├─checkpoints_hf
│  └─Llama-3.2-1B-hf
├─data
│  └─public_data
│      ├─dev
│      │  ├─track_a
│      │  ├─track_b
│      │  └─track_c
│      └─train
│          ├─track_a
│          └─track_b
└─output
```
